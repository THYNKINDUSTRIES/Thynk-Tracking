## Getting Started

Simply open `index.html` in any modern web browser. No installation, build process, or dependencies required!

## Data Storage

All data is stored locally in your browser using localStorage. **Important:** Export your data regularly to prevent data loss if you clear your browser data.

## Export Options

- **Export All Data** - Complete backup of all records
- **Chain of Custody Report** - Compliance report showing full traceability
- **Current Inventory Report** - Snapshot of current stock levels
- **Import Data** - Restore from previous export
